### react-koa-project

react+koa+mysql 开发的网站，

```bash
git clone git@github.com:chenjun1127/react-new-project.git
cd react-new-project
npm install or yarn
```

#### develop environment
```bash
npm run dev
```
#### production environment
```bash
npm run build
```
