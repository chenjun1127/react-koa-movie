## react-koa-project

react+mobx+koa+sequelize(mysql) 开发的网站，前端采用react全家桶，后端使用koa服务，数据库为mysql

### Install and run
```bash
git clone git@github.com:chenjun1127/react-koa-project.git
cd react-koa-project
npm install or yarn
npm run server 
npm run dev
```

### develop environment
```bash
npm run dev
```
### production environment
```bash
npm run build
```
